# AI 共學聚後端 — 啟動說明

## 第一步：安裝套件

```bash
cd ai-signup-backend
npm install
```

## 第二步：設定環境變數

複製 `.env.example` 並重命名為 `.env`，然後填入：

### Gmail 設定
1. 開啟 Google 帳號 → 安全性 → 兩步驟驗證（先開啟）
2. 搜尋「應用程式密碼」→ 建立新密碼
3. 把那 16 碼填入 `EMAIL_APP_PASSWORD`

### LINE Messaging API 設定
1. 前往 https://developers.line.biz/
2. 建立 Provider → 建立 Messaging API Channel
3. 取得 `Channel Secret` 和 `Channel Access Token`
4. 在 LINE Developers 設定 Webhook URL：
   `https://你的網域/webhook`
5. 啟用 Webhook，關閉自動回覆訊息

## 第三步：啟動伺服器

```bash
# 開發模式（自動重啟）
npm run dev

# 正式模式
npm start
```

伺服器啟動後：
- 表單送出端點：`http://localhost:3000/register`
- 後台管理：`http://localhost:3000/admin.html`
- LINE Webhook：`http://localhost:3000/webhook`

## 第四步：讓外部可以存取（部署）

本機測試可用 **ngrok**：
```bash
npx ngrok http 3000
```
把 ngrok 給的 https URL 貼到 LINE Developers 的 Webhook URL。

正式部署建議：**Railway** 或 **Render**（免費方案可用）

## LINE 綁定流程說明

1. 在報名表單放上 LINE 官方帳號的 QR Code
2. 報名者加入後，傳送他們的 Email 給 Bot
3. Bot 自動將 LINE User ID 與報名資料綁定
4. 活動前自動推送提醒
